## 🚨 Critical Errors to Avoid

❌ **DO NOT:**

- Edit HTML/CSS without knowing what you're doing (unless confident)
- Remove required sections
- Break the code structure (mismatched brackets)
- Use passwords that are too obvious
- Share without testing first
- Include real passwords or sensitive info
- Forget to save files after editing
- Use ALL CAPS in the love letter (hard to read)

✅ **DO:**

- Test changes before sharing
- Keep backup copies
- Make small changes at a time
- Save frequently (Ctrl+S)
- Hard refresh browser (Ctrl+Shift+R)
- Proofread your text
- Test on mobile
- Give yourself time to review
